h1 até h6

As tags h1 até h6 são usadas para definir cabeçalhos em um documento HTML. O h1 é o mais importante (maior) e o h6 é o menos importante (menor).

p

A tag p define um parágrafo de texto. É uma das tags mais utilizadas para estruturar o conteúdo textual em páginas web.

mark

A tag mark é usada para destacar texto, geralmente exibindo o fundo do texto em amarelo.

small

A tag small reduz o tamanho do texto, frequentemente usada para notas de rodapé ou detalhes menos importantes.

i

A tag i formata o texto em itálico, geralmente para destacar uma palavra ou frase.

u

A tag u sublinha o texto. É usada para destacar informações importantes ou links em alguns casos.

strong

A tag strong indica um texto com importância ou ênfase forte. Geralmente, o navegador exibe o texto em negrito.

ol

A tag ol cria uma lista ordenada, ou seja, numerada. Cada item da lista é definido com a tag li.

ul

A tag ul cria uma lista não ordenada, ou seja, com marcadores (bullets). Cada item da lista é definido com a tag li.

li

A tag li define um item em uma lista, que pode ser ordenada (ol) ou não ordenada (ul).

a

A tag a cria um link. Com o atributo "href", é possível direcionar o usuário para outra página ou recurso.

hr

A tag hr insere uma linha horizontal, usada para separar conteúdos de forma visual.

sub

A tag sub é usada para criar texto subscrito, como em fórmulas químicas (exemplo: H2O).

sup

A tag sup cria texto sobrescrito, usado frequentemente para expoentes (exemplo: 23).

blockquote

A tag blockquote é usada para definir uma citação em bloco. Normalmente, o texto aparece indentado.

TAGS Extras

  • font
  • del
  • abbr

font

A tag font; é usada para definir a aparência do texto, como cor, tamanho e tipo de fonte. É considerada obsoleta em HTML5, substituída pelo uso de CSS.

del

A tag del indica texto que foi removido ou descartado. Normalmente, o navegador exibe o texto riscado.

abbr

A tag abbr define abreviações ou acrônimos. O atributo "title" é usado para fornecer a explicação completa quando o usuário passa o cursor sobre o texto.
